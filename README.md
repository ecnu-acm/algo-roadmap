# ACM知识体系

## 数据结构
* **基础**：并查集（路径压缩/按秩合并）、树状数组（BIT）、ST表（Sparse Table）、单调队列/栈、Trie（字典树）
* **进阶**：线段树（延迟标记/合并/分裂）、平衡树（Treap / Splay / FHQ-Treap）、树链剖分（重链/长链）、分块、莫队算法（普通/树上/回滚）
* **高阶**：LCT（Link-Cut Tree）、主席树（可持久化线段树）、KD-Tree、析合树、笛卡尔树、跳表 (Skip List)、左偏树/配对堆（可合并堆）、李超线段树（处理线段覆盖极值）、线段树分治

##  图论
* **基础**：最短路（Dijkstra / Floyd）、Tarjan 算法（SCC/割点/桥）、最小生成树（Kruskal / Prim）、拓扑排序
* **进阶**：差分约束系统、二分图匹配（匈牙利算法）、欧拉回路/哈密顿路、虚树（处理关键点问题）、2-SAT
* **高阶**：网络流（Dinic / ISAP / 费用流 / 上下界流）、一般图匹配（带花树）、图论计数（Matrix-Tree 定理）、支配树、圆方树、朱刘算法（最小树形图）

## 动态规划
* **基础**：线性/背包/区间DP、数位DP、树上DP
* **进阶**：状压DP、斜率优化、四边形不等式优化、概率与期望DP、单调队列优化
* **高阶**：DP套DP、动态DP（DDP）、WQS 二分优化、插头 DP、多项式/生成函数卷积优化

## 字符串
* **基础**：KMP、字符串哈希、最小表示法
* **进阶**：AC自动机、Manacher算法、回文树（PAM）、Lyndon分解、广义 AC 自动机、Z 算法 (ExKMP)
* **高阶**：后缀自动机（SAM）、后缀数组（SA）、后缀树、本质不同子串计数、广义后缀自动机（处理多串问题）、后缀平衡树

## 数学
* **基础**：欧几里得算法（GCD）、扩展欧几里得（EXGCD）、费马小定理、排列组合、线性筛素数
* **进阶**：
  * 数论：中国剩余定理（CRT/EXCRT）、欧拉定理/欧拉函数、卢卡斯定理（Lucas）、博弈论（SG 函数）
  * 线性代数：线性基、矩阵乘法加速、高斯消元
  * 概率：容斥原理、斯特林数、卡特兰数
* **高阶**：
  * 筛法：杜教筛、Min_25 筛、莫比乌斯反演
  * 多项式：FFT/NTT/FWT（快速卷积家族）、多项式求逆/开方/对数/指数
  * 其他：二次剩余（Cipolla）、线性规划（单纯形法）、Pólya 计数、Burnside 引理

## 计算几何
* **基础**：向量运算（点积/叉积）、线段交点、多边形面积、点/线/圆位置关系
* **进阶**：二维凸包、旋转卡壳、半平面交、离散化扫描线、最小圆覆盖、辛普森积分
* **高阶**：动态凸包、闵可夫斯基和、三维几何（三维凸包）、KD-Tree 几何应用、纳皮尔圆周

## 杂项
* **基础**：二分/三分、离散化、暴力剪枝
* **进阶**：CDQ 分治、整体二分、点分治、根号分治、随机化（模拟退火）、交互题策略
* **高阶**：构造题专项、动态点分治（点分树）、边分治、DLX（舞蹈链）

## 后记

### 书籍
**《算法导论》**

**《算法竞赛进解指南》—— 李煜东**
> **地位：** 目前国内公认的“冲金首选”。 

> **特点：** 相比于入门书籍，这本书对算法的底层逻辑、数学证明和思维演练讲得非常透彻。书中的例题选自经典的区域赛真题，涵盖了大部分“进阶”到“高阶”的过渡内容。

**《算法竞赛入门经典》（紫书）及《算法竞赛入门经典：训练指南》（蓝书）—— 刘汝佳**
> **地位：** ACM 界的“圣经”。

> **特点：** 刘汝佳老师的选题极具代表性。蓝书《训练指南》深入讨论了计算几何、组合数学和高级数据结构，是冲击 WF 的必备练习册。

**《具体的数学：计算机科学基础》（Concrete Mathematics）—— Graham / Knuth / Patashnik**
> **领域：数学** 

> **特点：** 如果你想在数论、组合数学、生成函数上达到 WF 级别，这本书是绕不开的。它不是专门的竞赛书，但它教会你如何严谨地处理各种复杂的求和、递归和数论问题。

《Competitive Programming 4》（CP4）—— Steven Halim
> **地位：** 国际选手的通用教材。

> **特点：** 涵盖了大量 Codeforces 和 UVa 的真题分类，侧重于“题型识别”和“实战技巧”，对于团队配合和比赛策略也有独到的见解。

《Ants》（挑战程序设计竞赛）—— 秋叶拓哉等
> **地位：** 日本选手的神作。

> **特点：** 逻辑极其严密，代码精炼。书中对“建模”的过程描述得非常精彩，适合培养将实际问题转化为算法模型的直觉。


《ACM-ICPC世界总决赛试题解析》

《算法艺术与信息学竞赛》
